So You Want To Learn TypeScript?

Hello colleague, and welcome to the TypeScript tutorial!
In this tutorial, we'll explore the fundamentals of TypeScript step-by-step.

This guide will provide you with a solid foundation in TypeScript. We'll cover the core concepts, syntax, and features that make TypeScript such a valuable tool in modern web development, and the differences between it and JavaScript.

What you'll learn:

  • What is TypeScript?
  • Why use TypeScript?
  • How to set up a TypeScript project
  • Basic TypeScript syntax
  • Differences between TypeScript and JavaScript
  • How to convert a JavaScript project to TypeScript
  • Useful links and resources

Since the expectation is that you are already comfortable and have plenty of experience with JavaScript, we'll jump straight into the TypeScript tutorial below. If you're new to JavaScript, you may want to start with a JavaScript tutorial first to get a better understanding of the basics.

Basic Types

TypeScript allows you to define types for variables.

examples:


            let message: string = 'Hello, TypeScript!';
            let count: number = 5;
            let isActive: boolean = true;

Functions

You can define functions with type annotations to specify the types of parameters and return values.

examples:


            function add(a: number, b: number): number {
              return a + b;
            }
            let result: number = add(3, 5);

Interfaces

Interfaces define the shape of objects.

examples:


            interface Person {
              name: string;
              age: number;
            }
            let person: Person = { name: 'Eleanor', age: 25 };

Classes

TypeScript supports classes and object-oriented programming.

examples:


            class Animal {
              name: string;
              constructor(name: string) {
                this.name = name;
              }
              speak() {
                console.log(this.name + ' makes a noise.');
              }
            }
            let cat = new Animal('Kitty');
            cat.speak();
